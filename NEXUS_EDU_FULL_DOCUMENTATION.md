# 🌟 التوثيق الشامل لمنصة NEXUS EDU (نكسس التعليمية)

## 📌 1. مقدمة عن المشروع (فكرة المنصة)
**NEXUS EDU** (المعروفة سابقاً بـ Million Platform) هي منصة تعليمية ذكية ومتكاملة، تهدف إلى إحداث ثورة في تجربة التعليم المدرسي من خلال دمج تقنيات الذكاء الاصطناعي (AI)، وأنظمة التلعيب (Gamification)، وأدوات الأتمتة الإدارية.
توفر المنصة بيئة مترابطة تجمع بين **الطالب**، **المعلم**، **ولي الأمر**، و**الإدارة المدرسية** في مكان واحد لتقديم تجربة تعليمية ممتعة، تفاعلية، وقابلة للقياس.

---

## 🏗️ 2. المعمارية التقنية (Technical Architecture)
تم بناء المشروع باستخدام بنية الـ **Monorepo** باستخدام تقنية `Turborepo` لتسهيل إدارة الواجهة الأمامية والخلفية في مستودع واحد.

### 🌐 الواجهة الأمامية (Front-end - Web App)
* **إطار العمل:** Next.js 16 (App Router).
* **لغة البرمجة:** TypeScript.
* **التصميم:** TailwindCSS + Shadcn UI مع دعم الوضع الليلي وتصميمات زجاجية (Glassmorphism).
* **إدارة الحالة والبيانات:** React Query & Zustand.
* **دعم اللغات:** Next-Intl (لدعم اللغتين العربية والإنجليزية).

### ⚙️ الواجهة الخلفية (Back-end - API)
* **إطار العمل:** NestJS (بنية قوية قابلة للتوسع).
* **قاعدة البيانات:** PostgreSQL (مستضافة على Supabase).
* **إدارة قواعد البيانات (ORM):** Prisma v6.
* **الاتصال اللحظي (Real-time):** Socket.IO (للمحادثات والإشعارات).
* **الأمان:** JWT Authentication, Rate Limiting, Helmet.

## 🚀 3. المميزات الأساسية (Core Features)

### 🧑‍🎓 بوابة الطالب (Student Portal)
1. **نظام التلعيب (Gamification):** رحلة "المليون" (Million Journey)، نقاط خبرة (XP)، مستويات، إنجازات (Badges)، ولوحة الصدارة (Leaderboard).
2. **المعلم الذكي (AI Tutor):** مساعد ذكاء اصطناعي يشرح الدروس، يحلل نقاط الضعف، ويقدم محتوى مخصص بناءً على مستوى الطالب (Smart Profile).
3. **الواجبات والدروس:** عرض الواجبات، تسليمها، وتتبع نسبة الإنجاز.
4. **الألعاب التعليمية:** بيئة تفاعلية لتعلم المواد عن طريق اللعب والمنافسة.

### 👨‍🏫 بوابة المعلم (Teacher Portal)
1. **منشئ المحتوى بالذكاء الاصطناعي (AI Content Generator):** أداة لتوليد خطط الدروس، بنوك الأسئلة، والواجبات بضغطة زر.
2. **أتمتة التصحيح (Auto-Grading):** تصحيح الواجبات تلقائياً وإعطاء تغذية راجعة (Feedback) فورية للطالب.
3. **الحضور بالـ QR (QR Attendance):** نظام سريع لتحضير الطلاب باستخدام مسح كود QR في الفصل.
4. **تحليل المخاطر (Student Risk Report):** لوحة بيانات ترصد الطلاب المعرضين للتراجع الأكاديمي بناءً على غيابهم ودرجاتهم.

### 👨‍👩‍👦 بوابة ولي الأمر (Parent Portal)
1. **المستشار الذكي لولي الأمر (AI Parent Advisor):** ذكاء اصطناعي يقرأ بيانات الطالب وينصح ولي الأمر بكيفية مساعدة ابنه في المنزل.
2. **متابعة حية:** تتبع حضور وغياب الطالب، الواجبات المتأخرة، والمستوى الأكاديمي.
3. **التقارير الدورية:** استلام تقارير تلقائية مفصلة (Scheduled Reports).

### 🏛️ بوابة الإدارة (Admin & School Management)
1. تشمل أدواراً متعددة: (مدير، وكيل، مرشد طلابي، مشرف تربوي، محاسب).
2. إدارة الجداول الدراسية، الفصول، والمعلمين.
3. تحليلات شاملة للمدرسة لتقييم الأداء العام.

## 🛠️ 4. رحلة التطوير وما قمنا بإنجازه (Timeline)

1. **تصميم وتطوير الواجهات:** قمنا بتطوير أكثر من 5 بوابات مختلفة بتصميم عصري (UI/UX) يحمل هوية NEXUS EDU.
2. **تطوير الباك إند وتصميم الجداول:** صممنا `schema.prisma` ضخمة ومعقدة تربط كل عناصر المدرسة (المواد، الحصص، التقييم، التلعيب).
3. **دمج الذكاء الاصطناعي:** ربطنا الـ API مع نماذج `OpenAI` لبناء الذكاء الاصطناعي الخاص بالمعلم والطالب.
4. **إلغاء البيانات الوهمية (Mock Data):** استبدلنا كل البيانات الوهمية في النظام بربط حقيقي بقاعدة البيانات.
5. **الرفع على سيرفرات الإنتاج (Deployment):**
   * تم رفع الـ API والـ Web بشكل منفصل على منصة **Railway**.
   * تم ربط قاعدة البيانات بخوادم **Supabase**.
   * قمنا بحل جميع مشاكل التعارض (مثل تعارض `middleware.ts` في Next.js).
   * قمنا بتهيئة متغيرات البيئة (Environment Variables) وتخطي تعقيدات الـ `CORS` والـ `Prisma Connection Pooler` ليعمل السيرفر بكفاءة 100%.
